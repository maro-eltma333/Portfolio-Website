## 🖼️ Features
- 🎨 Modern, responsive UI (Tailwind CSS + AOS animations)
- 🧩 Custom portfolio grid with an immersive, fully-responsive lightbox
- ✉️ Secure server-side contact form with **Resend API** + SweetAlert2 feedback
- 📱 Perfect mobile-first responsive navigation with modern active states
- ⚡ Optimized Next.js App Router architecture

## 🧰 Tech Stack
- **Framework**: Next.js (App Router), React
- **Styling**: Tailwind CSS
- **Interactions**: AOS, SweetAlert2, Lucide React
- **Backend/API**: Next.js Server Actions, Resend API (Email Delivery)
- **Language**: TypeScript

## 📁 Project Structure
```text
Portfolio-Website/
├─ public/               # Static assets (images, SVGs)
├─ src/
│  ├─ app/
│  │  ├─ actions.ts      # Server Actions (Resend Email API logic)
│  │  ├─ globals.css     # Global styles & Tailwind configuration
│  │  ├─ layout.tsx      # Next.js Root Layout
│  │  └─ page.tsx        # Main UI (Nav, Portfolio, Lightbox, Contact)
├─ next.config.ts        # Next.js Configuration
├─ package.json          # Project Dependencies
└─ README.md             # You are here 💡
```
